# Encurtador de Url

## Descrição<a name="descricao"></a>
Projeto em node para encurtar URL e redirecionar para a URL original.
Feito e arquiteturado para rodar em um ecosistema de microserviços.


## Tecnologias<a name="tecnologias"></a>
#### Runtime
  * Node 14
  * Express
  * Nestjs
  * Postgres
  * Cache in-memory
